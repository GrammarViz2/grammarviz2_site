Time series anomaly and recurrent patterns discovery made interactive

GrammarViz approach is based on two techniques with a linear time and space complexity: Symbolic Aggregate approXimation (SAX), that discretizes the input time series into a string, and Sequitur, that induces a context-free grammar (CFG) from it.

By exploiting the hierarchical structure of CFG, GrammarViz is able to instantly* identify rare and frequent grammar rules, which we associate with anomalous and recurrent subsequences. In addition, thanks to SAX numerosity reduction and CFG hierarchy, our approach is able to discover patterns of both types that are of variable length in a single run.

* note that the patterns discovery through grammatical inference involves no explicit distance computation
Please find details about our technique here: GrammarViz 2.0: a tool for grammar-based pattern discovery in time series.
Senin, P., Lin, J., Wang, X., Oates, T., Gandhi, S., Boedihardjo, A.P., Chen, C., Frankenstein, S., Lerner, M., ECML/PKDD, 2014.

User and developer friendly platform

GrammarViz 2.0 is developed in Java and can be used interactively through the GUI, called from a command line, or linked to your code as a library. We followed the Model–view–controller (MVC) pattern in GUI development which allows for code re-use in a web applicaion. For your convenience, our code is hosted at GitHub, we do Unit test it, analyze with FindBugs and PMD, and use Travis CI for our builds.

Roadmap

Our software is under active development. Among other things, we are investigating alternative GI algorithms performance, researching the possibility to leverge a grammar's hierarchy for patterns weighting, and working on the system performance and usability.
